## ملاحظات مهمة - Important Notes

1. **جميع الطلاب ناجحون**: كلا الطالبين حصلا على درجة "ناجح" في جميع المواد
2. **المعدل منخفض**: كلا الطالبين لديهما معدل أقل من 2.0 مما يضعهما في حالة "تحذير"
3. **الساعات المعتمدة**: كل مادة تحمل 3 ساعات معتمدة
4. **إجمالي الساعات**: 21 ساعة معتمدة لكل طالب ف�� هذا الفصل

## كيفية إضافة الطلاب - How to Add Students

### الطريقة الأولى: إعادة تشغيل جميع البذور
```bash
php artisan migrate:fresh --seed
```

### الطريقة الثانية: إضافة الطلاب الجدد فقط
```bash
php artisan db:seed --class=NewStudentsSeeder
```

### الطريقة الثالثة: استخدام الملف المساعد
```bash
add_new_students.bat
```
