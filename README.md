# بوابة طلاب الجامعة الأردنية - Jordan University Student Portal

نسخة مطورة من بوابة طلاب الجامعة الأردنية باستخدام Laravel مع دعم كامل للغة العربية ونظام إدارة النتائج الأكاديمية.

## المميزات

- **واجهة مستخدم عربية**: تصميم متجاوب يدعم اللغة العربية بالكامل
- **نظام المصادقة**: تسجيل دخول آمن للطلاب
- **إدارة النتائج الأكاديمية**: عرض النتائج والدرجات لجميع الفصول
- **حساب المعدلات**: حساب تلقائي للمعدل الفصلي والتراكمي
- **إحصائيات أكاديمية**: عرض الساعات المنجزة ومؤشر الفصل
- **المواد المكافئة**: نظام إدارة المواد المكافئة لكلية الحقوق
- **تصميم متجاوب**: يعمل على جميع الأجهزة والشاشات

## متطلبات النظام

- PHP 8.0 أو أحدث
- Composer
- MySQL 5.7 أو أحدث
- Node.js و npm (اختياري للتطوير)

## التثبيت والإعداد

### 1. تحضير قاعدة البيانات

```bash
# إنشاء قاعدة بيانات جديدة
mysql -u root -p
CREATE DATABASE ju_student_portal CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2. تكوين متغيرات البيئة

```bash
# نسخ ملف البيئة
cp .env.example .env

# تحرير ملف .env وتحديث إعدادات قاعدة البيانات
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ju_student_portal
DB_USERNAME=root
DB_PASSWORD=your_password
```

### 3. تثبيت التبعيات

```bash
# تثبيت تبعيات PHP
composer install

# إنشاء مفتاح التطبيق
php artisan key:generate
```

### 4. تشغيل الترحيلات والبذور

```bash
# تشغيل ترحيلات قاعدة البيانات
php artisan migrate

# إدراج البيانات التجريبية
php artisan db:seed
```

### 5. تشغيل الخادم

```bash
# تشغيل خادم التطوير
php artisan serve
```

الآن يمكنك الوصول للتطبيق عبر: `http://localhost:8000`

## هيكل قاعدة البيانات

### الجداول الرئيسية:

- **students**: معلومات الطلاب الأساسية
- **courses**: المواد الدراسية
- **enrollments**: تسجيل الطلاب في المواد والدرجات
- **academic_records**: السجلات الأكاديمية الفصلية
- **equivalent_courses**: المواد المكافئة

### العلاقات:

- طالب واحد له عدة تسجيلات (One-to-Many)
- مادة واحدة لها عدة تسجيلات (One-to-Many)
- طالب واحد له عدة سجلات أكاديمية (One-to-Many)
- مادة واحدة لها عدة مواد مكافئة (One-to-Many)

## حساب المعدلات والإحصائيات

### المعدل الفصلي
```
المعدل الفصلي = مجموع (النقاط × الساعات) ÷ مجموع الساعات الفصلية
```

### المعدل التراكمي
```
المعدل التراكمي = مجموع جميع (النقاط × الساعات) ÷ مجموع جميع الساعات
```

### مؤشر الفصل
- **ممتاز**: المعدل الفصلي ≥ 3.75
- **شرف**: المعدل الفصلي ≥ 3.5
- **إنذار أكاديمي**: المعدل التراكمي < 2.0
- **تحذير**: المعدل التراكمي < 2.5
- **منتظم**: باقي الحالات

## التطوير والمساهمة

### إضافة طالب جديد

```php
Student::create([
    'student_id' => '20210004',
    'name' => 'اسم الطالب',
    'email' => 'student@ju.edu.jo',
    'password' => Hash::make('password'),
    'college' => 'كلية الحقوق',
    'major' => 'القانون',
    'academic_year' => '2021/2022'
]);
```

### إضافة مادة جديدة

```php
Course::create([
    'course_code' => '0803500',
    'course_name' => 'Course Name',
    'course_name_ar' => 'اسم المادة',
    'credit_hours' => 3,
    'college' => 'كلية الحقوق',
    'department' => 'القانون'
]);
```

### تسجيل طالب في مادة

```php
Enrollment::create([
    'student_id' => 1,
    'course_id' => 1,
    'academic_year' => '2021',
    'semester' => 2,
    'grade' => 'A',
    'grade_points' => 4.0,
    'is_passed' => true
]);
```
